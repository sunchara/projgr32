https://github.com/sunchara/projgr32/blob/f97454583e0606e25e0cc934ccafaafe248c5664/plane/plane_ICS.cpp#L217

1)изменение точки просмотра - "перетаскиванием картинки"
2)колесиком мыши - увеличить / уменьшить
